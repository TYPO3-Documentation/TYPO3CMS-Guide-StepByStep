# Add a Regular Text Element
**Tested in:** [TYPO3v13](/Tags/TYPO3v13.md)
**Categories:** [Beginner](/Tags/Beginner.md) [Editing](/Tags/Editing.md) [ContentElements](/Tags/ContentElements.md)
**Author:** [@csabareanu](https://my.typo3.org/u/csabareanu)

Text content is the most common building block on TYPO3 websites.

## Learning objective

Add and publish a text content element on a page.

## Add a text content element

1. Open the **Page** module.
2. Select a page.
3. Click **+ Create new content**.
4. Choose **Regular Text Element**.
<img width="991" height="486" alt="image" src="https://github.com/user-attachments/assets/fcab0a2f-9f58-47fb-92b2-8ce44b0b0cfc" />


## Enter and format text

1. Enter your content in the Text field.
2. Use the toolbar to apply basic formatting (headings, bold/italic, lists). [Format Text and Add Links with the Rich Text Editor](/10GettingStarted/30ContentCreation/30WorkWithTheRichTextEditor/FormatTextAndAddLinksWithTheRte.md)
3. Add links if needed.

## Save and verify

1. Click Save (or Save and close).
2. Use the preview (or open the page in the frontend) to verify the content appears correctly.

## Summary

You created a text content element, formatted it with the RTE, and saved it so it is visible on the page.
